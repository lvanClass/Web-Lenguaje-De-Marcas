## Expresiones Regulares para la Validación

### Validación de Contraseña
- **Expresión Regular**: `^(?=.*[A-Za-z])(?=.*\d)(?=.*[@$!%*#?&])[A-Za-z\d@$!%*#?&]{8,}$`
- **Explicación**: 
  - Mínimo 8 caracteres
  - Al menos una letra
  - Al menos un número
  - Al menos un carácter especial
